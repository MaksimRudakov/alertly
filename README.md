# alertly Helm chart repository

This branch hosts the Helm chart index for [alertly](https://github.com/MaksimRudakov/alertly).

```
helm repo add alertly https://maksimrudakov.github.io/alertly
helm repo update
helm search repo alertly
```

Managed automatically by [`helm/chart-releaser-action`](https://github.com/helm/chart-releaser-action) — do not edit by hand.
